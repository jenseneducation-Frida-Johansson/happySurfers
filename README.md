# vue-coffe-shop

1. install npm:
   npm install
2. install axios:
   npm install --save axios vue-axios
3. cd airbean-api-master
4. install cors:
   npm install cors
5. run backend server:
   node app.js
6. open another terminal, open 'happysurfers' again:
   cd happysurfers
7. run server:
   npm run serve
